Spowerfy


Spowerfy is a web app I wrote with Python using the micro web framework Flask. Spowerfy authenticates with the user's Spotify account and controls the playback of whichever playlist the user configures it to. It does this all while updating the album art, current track, and current artist in lockstep with the playback.
